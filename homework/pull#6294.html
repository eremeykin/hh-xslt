This PR is an enhancement to the documentation and website of CRA.
I've been recently working on Documate - a tool built with CRA that pretty much does what Docusaurus does, just a bit better. It generates a React-like site design out of Markdown files.

Within this PR I simply migrate CRA's site/docs from Docusaurus to Documate. There's not much change, however I've cleaned up the doc files a bit. However there's a major UI difference since Documate tries to better resemble ReactJS's site and docs.
